## Python 3 a Dark Template

Features:

- Frameless
- Dragable and resizeable
- full QSS included, based on QSS from [QDarkStyleSheet](https://github.com/ColinDuquesnoy/QDarkStyleSheet)
- included SQLite Example
- QSS for a Table
